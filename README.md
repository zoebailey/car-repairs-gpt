# car-repairs-gpt
Visual learner? Don't understand long car manuals? Use ChatGPT's gpt-image-1.5 to create visual instructions for repairs! 

## Workflow
- choose what type of repair is needed
- app generates steps aided by large images and short descriptions

## What this is
- a prototype that uses OpenAI’s Image Gen 1.5 API to turn car repair steps into visual information

## Why it’s interesting
- text-only manuals are hard to follow, especially for beginners

## How to run
- clone repo
- install dependencies
- set OPENAI_API_KEY
- run script
